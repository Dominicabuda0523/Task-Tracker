# Task Tracker
Simple Task Tracker, a project suggestion at https://roadmap.sh/projects/task-tracker.

Welcome to the Task Tracker CLI!

Available commands:
1. add <description> - Add a new task with the given description.
   
3. delete <id1>, <id2>, ... - Delete tasks by their IDs (comma-separated).
   
5. update <id>, <new description> - Update the description of a task by its ID.
   
7. mark <id>, <status> - Mark a task with a new status (Todo, In-Progress, Done).
   
9. list <status> - List tasks by status (Todo, In-Progress, Done) or list all tasks with 'list all'.
    
11. exit - Exit the program.


P.S: Note that in the first time running the code, it is necessary to add a task first before trying out all the other commands or else it will error. Also open for criticism (^_^) for the betterment as a learner.
