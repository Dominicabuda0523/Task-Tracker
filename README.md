# Task Tracker
Simple Task Tracker, a project suggestion at https://roadmap.sh/projects/task-tracker.

Running the code, you will see a simplified interface in which you could select actions by typing the corresoponding number.
Entering (1) See List - will list all tasks that is saved in the json file and you will also be given an another commands that will let you see depending on the status of a task, (list done)(list in-progress)(list todo)(go back). All of which are not case sensitive.

Entering (2) Add Task - you will be directed into an input function in where you input a task. After creating one, you will return to the interface.

Entering (3) Delete Task - all the tasks will be listed in the CLI, including all of their properties, in which you could delete a task by inputing the ID of a specific task. After deleting, you will be directed into the interface.

Entering (4) Update Task - all the tasks will be listed again and you will input their own id to select and change their status. You can change it by inputing (todo)(in-progress)(done), it is also not case sensitive.

Entering (5) Quit - will terminate the code.

P.S: Note that in the first time running the code, it is necessary to add a task first before trying out all the other commands or else it will error. Also open for criticism (^_^) for the betterment as a learner.
